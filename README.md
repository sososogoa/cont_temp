# Cont_Temp

임시 저장소

## 사용 기술 스택

- **React**
- **TypeScript**
- **TailwindCSS**
- **Radix UI**
- **Framer Motion**
- **Prisma ORM**
- **PostgreSQL**
- **Node.js & Express**
- **Redux**

## 임시 기능

- `http://localhost:3000/api/example`에서 API 테스트 가능.
- `main.tsx`에서 Redux 설정을 통해 카운터 기능

## 시작하기

1. 저장소를 클론합니다:
   ```bash
   git clone https://github.com/sososogoa/cont_temp.git
   ```
